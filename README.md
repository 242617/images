# images

## go-builder

Alpine-based image for building Golang 1.13 applications.

## Usage

|             Command             |     Usage   |
|:--------------------------------|:-------------|
| Build image                     | `make build` |
| Test image (interactive loging) | `make test`  |
| Save image to file              | `make save`  |
| Push image to hub               | `make push`  |
